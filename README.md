## Use locally

```bash
git clone https://github.com/alassaneba1/dev-learning-lab.git
cd dev-learning-lab
# Open index.html in your browser — no server needed
```

---

## Features

- Dark / light theme with persistent toggle
- Search across all sections
- Syntax-highlighted, copy-ready code examples
- Self-contained HTML — works offline

## Contributing

Want to add a cheatsheet?

1. Copy an existing file in `cheatsheets/` as a starting point
2. Add your entry to the table in `index.html` and this README
3. Open a PR
